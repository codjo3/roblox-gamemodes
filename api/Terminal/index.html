Terminal

A terminal (control-point capture) gamemode built on GamemodeCore.

Teams compete to hold a BasePart zone long enough to accumulate points. The first team to reach PointsNeeded wins. Supports overtime, stacking capture speed, defender win conditions, and React-style hooks via the inherited mount API. 

local Terminal = require(Packages.GamemodeTerminal)

local gm = Terminal.new("RoundOne", {
    Teams = { game.Teams.Red, game.Teams.Blue },
    Parts = { workspace.ControlPoint },
    PointsNeeded = 1200,
    CaptureTime = 10,
})

gm.Stopped:Connect(function(winner)
    print(winner.Name .. " wins!")
end)

gm:Start()

Settings​

</>
interface Settings {
Teams: {Team}--

Teams competing in the round. Required.

Parts: {BasePart}--

Parts that define the capture zone. Required.

Increase: number?--

Points per second earned while holding. Defaults to 1.

Rollback: number?--

Points per second lost while not holding. Defaults to 1.

PointsNeeded: number?--

Points required to win. Defaults to 1200.

RequireDominanceToIncrease: boolean?--

Only earn points when opponent has zero. Defaults to true.

ProgressWhileContested: boolean?--

Allow capture progress when both teams are present. Defaults to true.

CaptureWhileDead: boolean?--

Count dead players toward zone occupation. Defaults to false.

CaptureStacking: boolean?--

Scale capture speed by player count difference. Defaults to false.

CaptureTime: number?--

Seconds to fully capture the point. Defaults to 1.

OvertimeThreshold: number?--

Elapsed seconds before overtime begins. -1 disables. Defaults to -1.

OvertimeIncrease: number?--

Points per second during overtime. Defaults to 1.

OvertimeRollback: number?--

Rollback rate during overtime. Defaults to 1.

OvertimePointsNeeded: number?--

Points to win during overtime. Defaults to 120.

OvertimeDefenderWinFullDominance: boolean?--

Defenders win if fully captured while opponent has zero. Defaults to true.

DefendingTeam: Team?--

The defending team for overtime. Defaults to Teams[1].

}

Configuration passed to Terminal.new.
