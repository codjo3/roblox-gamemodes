GamemodeCore

Reactive hook-based gamemode lifecycle core.

Register a gamemode type with GamemodeCore.register, then create instances with GamemodeCore.new. Each instance manages an Active/Paused state machine, a per-frame tick loop, and a React-style hook system via mount. 

local GamemodeCore = require(Packages.GamemodeCore)

GamemodeCore.register("MyMode", function(self, dt)
    -- custom per-frame logic
end)

local gm = GamemodeCore.new("MyMode", "RoundOne", {
    ScoreLimit = 100,
})

gm.Stopped:Connect(function(winner, metadata)
    print("Winner:", winner)
end)

gm:Start()

Registers a gamemode type with a tick function. Must be called before GamemodeCore.new for that type. Registering the same type twice overwrites the previous registration with a warning. 

GamemodeCore.register("MyMode", function(self, dt)
    if self.Elapsed > 300 then
        self:Stop(nil)
    end
end)

mount​

</>
GamemodeCore:mount(
fn: (
gm: GamemodeCore,
ctx: HookContext
) → ()--

Reactive function to mount.

) → () → ()--

Call to unmount and clean up.

Attaches a reactive function to the gamemode. The function receives the instance and a HookContext and is called every Heartbeat while the gamemode is active and unpaused.

Returns an unmount function. Call it to detach the mount and run any pending useEffect cleanups. 

local unmount = gm:mount(function(self, ctx)
    local score, setScore = ctx:useState(0)

    ctx:useEffect(function()
        local c = self.Stopped:Connect(function() setScore(0) end)
        return function() c:Disconnect() end
    end, {})
end)

Start​

</>
GamemodeCore:Start(
metadata: any?--

Optional value forwarded to Started listeners.

) → ()

Starts the gamemode. Transitions Active to true, resets Elapsed to 0, begins the Heartbeat loop, ticks all mounted functions once, then fires Started.

Does nothing if already active.

Stop​

</>
GamemodeCore:Stop(
winner: any?,--

Passed as the first argument to Stopped listeners.

metadata: any?--

Passed as the second argument to Stopped listeners.

) → ()

Stops the gamemode. Disconnects the Heartbeat loop, flushes all useEffect cleanups, fires Stopped, then resets Elapsed to 0.

Does nothing if not active.

Pause​

</>
GamemodeCore:Pause(
metadata: any?--

Optional value forwarded to OnPaused listeners.

) → ()

Pauses the gamemode, suspending the tick loop and mount calls. Fires OnPaused. Does nothing if not active or already paused.

Resume​

</>
GamemodeCore:Resume() → ()

Resumes a paused gamemode. Does nothing if not active or not paused.

Destroy​

</>
GamemodeCore:Destroy() → ()

Destroys the gamemode. Fires Destroying, stops the round if active, destroys all mounts and signals, and removes the instance from the global registry.

After calling this, the instance must not be used again.
