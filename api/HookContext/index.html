HookContext

Provides React-style hooks to functions mounted onto a GamemodeCore instance via GamemodeCore:mount. A new HookContext is created per mounted function and persists for the lifetime of that mount.

Hooks must always be called in the same order on every invocation — the same rules as React hooks apply.

Returns the current value of a persistent state slot and a Dep that can be called to update it. On subsequent calls the initial value is ignored and the stored value is returned. 

gm:mount(function(self, ctx)
    local score, setScore = ctx:useState(0)
    -- call setScore(n) to update
end)

initial: T--

Initial state value.

) → (
T,--

Current state.

Dep<A>--

Callable dispatcher.

)

Like useState but the setter accepts an action that is passed through a reducer function to produce the next state. 

local function reducer(state, action)
    if action == "increment" then return state + 1 end
    return state
end

gm:mount(function(self, ctx)
    local count, dispatch = ctx:useReducer(reducer, 0)
    dispatch("increment")
end)

useMemo​

</>
HookContext:useMemo(
fn: () → T,--

Function that computes the value.

deps: {any}--

List of values to watch for changes.

) → T--

Cached computed value.

Returns a cached value that is only recomputed when one of the deps changes between calls. 

gm:mount(function(self, ctx)
    local expensive = ctx:useMemo(function()
        return computeSomething(self.Score)
    end, { self.Score })
end)

Function to memoize.

) → T--

Stable function reference.

Returns a memoized function reference that only changes when one of the deps changes. Equivalent to useMemo(function() return fn end, deps).

useRef​

</>
HookContext:useRef(
initial: T--

Initial value of .current.

) → Ref<T>

Returns a persistent mutable ref object with a .current field. Unlike useState, mutating .current does not trigger a re-run. 

gm:mount(function(self, ctx)
    local timerRef = ctx:useRef(0)
    timerRef.current += dt
end)

useEffect​

</>
HookContext:useEffect(
fn: () → (() → ())?,--

Effect function. Return a cleanup function or nothing.

deps: {any}--

Re-runs the effect when any dep changes.

) → ()

Runs a side effect when deps change. The effect function may return a cleanup function that is called before the next run or when the mount is destroyed. 

gm:mount(function(self, ctx)
    ctx:useEffect(function()
        local c = self.Stopped:Connect(function()
            print("stopped")
        end)
        return function() c:Disconnect() end
    end, {})
end)
